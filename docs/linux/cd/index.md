# cd 命令

## 命令简介
`cd` 命令用于切换当前工作目录，是 Linux 系统中最常用的导航命令。

## 语法
```bash
cd [目录路径]
```

## 常用用法
- `cd`：切换到当前用户的主目录
- `cd ~`：切换到当前用户的主目录（与 `cd` 相同）
- `cd -`：切换到上一个工作目录
- `cd ..`：切换到父目录
- `cd ../..`：切换到上两级目录
- `cd /`：切换到根目录

## 示例
1. 切换到主目录：
   ```bash
   cd
   ```

2. 切换到指定目录：
   ```bash
   cd /home/user/documents
   ```

3. 切换到父目录：
   ```bash
   cd ..
   ```

4. 切换到上一个工作目录：
   ```bash
   cd - 
   ```

5. 切换到根目录：
   ```bash
   cd /
   ```

## 注意事项
- `cd` 是 shell 内置命令，不是独立的可执行文件
- 使用绝对路径（以 / 开头）或相对路径（相对于当前目录）
- 可以使用环境变量作为目录路径，如 `cd $HOME`
