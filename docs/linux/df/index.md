# df 命令

## 命令简介
`df` 命令用于显示文件系统的磁盘空间使用情况，是 "disk free" 的缩写。

## 语法
```bash
df [选项] [文件/目录...]
```

## 常用选项
- `-h`：以人类可读的格式显示大小（如 K、M、G）
- `-a`：显示所有文件系统，包括虚拟文件系统
- `-T`：显示文件系统类型
- `-i`：显示inode使用情况
- `-k`：以KB为单位显示
- `-m`：以MB为单位显示
- `-P`：使用POSIX格式输出

## 示例
1. 显示磁盘空间使用情况：
   ```bash
   df
   ```

2. 以人类可读格式显示：
   ```bash
   df -h
   ```

3. 显示文件系统类型：
   ```bash
   df -T
   ```

4. 显示inode使用情况：
   ```bash
   df -i
   ```

5. 显示指定目录所在文件系统的使用情况：
   ```bash
   df -h /home
   ```

## 输出解释
主要字段：
- `Filesystem`：文件系统
- `Size`：总大小
- `Used`：已使用大小
- `Avail`：可用大小
- `Use%`：使用率
- `Mounted on`：挂载点

## 注意事项
- 默认以KB为单位显示
- `-h` 选项使输出更易读
- 可以查看指定文件或目录所在文件系统的使用情况
- 虚拟文件系统（如 proc、sysfs）的大小通常为0
