# cp 命令

## 命令简介
`cp` 命令用于复制文件或目录，是 "copy" 的缩写。

## 语法
```bash
cp [选项] 源文件 目标文件
cp [选项] 源文件... 目标目录
```

## 常用选项
- `-i`：交互式复制，覆盖前提示确认
- `-f`：强制复制，覆盖已存在的文件
- `-r` 或 `-R`：递归复制目录及其内容
- `-v`：显示复制的详细信息
- `-p`：保留文件的属性（权限、时间戳等）
- `-a`：归档复制，相当于 `-dpR`，保留所有属性

## 示例
1. 复制单个文件：
   ```bash
   cp file.txt file_copy.txt
   ```

2. 复制文件到目录：
   ```bash
   cp file.txt /home/user/documents/
   ```

3. 递归复制目录：
   ```bash
   cp -r dir1 dir2
   ```

4. 交互式复制：
   ```bash
   cp -i file.txt /home/user/
   ```

5. 保留属性复制：
   ```bash
   cp -p file.txt backup/
   ```

## 注意事项
- 复制目录必须使用 `-r` 或 `-R` 选项
- 默认情况下，复制的文件会继承目标目录的权限
- 使用 `-a` 选项可以保留文件的所有属性，包括符号链接
- 可以使用通配符复制多个文件，如 `cp *.txt backup/`
