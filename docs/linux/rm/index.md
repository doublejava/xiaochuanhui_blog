# rm 命令

## 命令简介
`rm` 命令用于删除文件或目录，是 "remove" 的缩写。

## 语法
```bash
rm [选项] 文件/目录...
```

## 常用选项
- `-f`：强制删除，不提示确认
- `-i`：交互式删除，删除前提示确认
- `-r` 或 `-R`：递归删除目录及其内容
- `-v`：显示删除的详细信息

## 示例
1. 删除单个文件：
   ```bash
   rm file.txt
   ```

2. 强制删除文件：
   ```bash
   rm -f file.txt
   ```

3. 递归删除目录及其内容：
   ```bash
   rm -r directory
   ```

4. 强制递归删除目录：
   ```bash
   rm -rf directory
   ```

5. 交互式删除多个文件：
   ```bash
   rm -i file1.txt file2.txt file3.txt
   ```

## 注意事项
- `rm` 命令删除的文件通常无法恢复，使用时需谨慎
- 删除目录必须使用 `-r` 或 `-R` 选项
- `-rf` 组合选项非常危险，可能导致数据丢失
- 建议使用 `-i` 选项进行交互式删除，避免误删
- 可以使用通配符删除多个文件，如 `rm *.txt`
