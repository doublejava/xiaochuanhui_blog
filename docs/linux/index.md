# Linux 常用命令

欢迎来到 Linux 常用命令教程！这里收录了 20 个最常用的 Linux 命令，每个命令都有详细的介绍、语法、选项、示例和注意事项。

## 命令列表

- [ls](./ls/) - 列出目录内容
- [cd](./cd/) - 切换当前工作目录
- [pwd](./pwd/) - 显示当前工作目录
- [mkdir](./mkdir/) - 创建新目录
- [rm](./rm/) - 删除文件或目录
- [cp](./cp/) - 复制文件或目录
- [mv](./mv/) - 移动或重命名文件和目录
- [cat](./cat/) - 查看文件内容
- [less](./less/) - 分页查看文件内容
- [head](./head/) - 显示文件开头部分
- [tail](./tail/) - 显示文件末尾部分
- [grep](./grep/) - 搜索文件内容
- [find](./find/) - 查找文件和目录
- [chmod](./chmod/) - 修改文件权限
- [chown](./chown/) - 修改文件所有者和所属组
- [ps](./ps/) - 显示进程状态
- [top](./top/) - 动态监控进程
- [kill](./kill/) - 终止进程
- [df](./df/) - 显示磁盘空间使用情况
- [du](./du/) - 显示目录或文件的磁盘使用情况

## 学习建议

1. 从基础命令开始学习，如 `ls`、`cd`、`pwd`
2. 每个命令都尝试运行示例，加深理解
3. 结合实际需求使用命令，如查看日志使用 `tail -f`
4. 学习使用命令的选项，提高工作效率
5. 多使用管道 `|` 组合命令，实现更复杂的功能

## 常用快捷键

- `Tab`：自动补全命令或文件名
- `Ctrl+C`：终止当前命令
- `Ctrl+Z`：暂停当前命令
- `Ctrl+D`：退出当前会话
- `Ctrl+L`：清屏
- `Ctrl+R`：搜索历史命令
