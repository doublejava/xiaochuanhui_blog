# pwd 命令

## 命令简介
`pwd` 命令用于显示当前工作目录的绝对路径，是 "print working directory" 的缩写。

## 语法
```bash
pwd [选项]
```

## 常用选项
- `-L`：显示逻辑路径（默认）
- `-P`：显示物理路径，解析所有符号链接

## 示例
1. 显示当前工作目录：
   ```bash
   pwd
   ```

2. 显示物理路径（解析符号链接）：
   ```bash
   pwd -P
   ```

3. 显示逻辑路径：
   ```bash
   pwd -L
   ```

## 注意事项
- `pwd` 是 shell 内置命令
- 输出的路径总是绝对路径（以 / 开头）
- 常用于脚本中获取当前目录信息
- 结合其他命令使用，如 `cd $(pwd)/subdir`
