# du 命令

## 命令简介
`du` 命令用于显示目录或文件的磁盘使用情况，是 "disk usage" 的缩写。

## 语法
```bash
du [选项] [文件/目录...]
```

## 常用选项
- `-h`：以人类可读的格式显示大小（如 K、M、G）
- `-s`：显示总计大小，不显示子目录
- `-a`：显示所有文件和目录的大小
- `-c`：显示总计大小
- `-d`：指定显示的深度，如 `-d 2` 显示两级目录
- `-k`：以KB为单位显示
- `-m`：以MB为单位显示

## 示例
1. 显示当前目录的磁盘使用情况：
   ```bash
   du
   ```

2. 以人类可读格式显示目录大小：
   ```bash
   du -h
   ```

3. 显示目录的总计大小：
   ```bash
   du -sh directory/
   ```

4. 显示两级目录的大小：
   ```bash
   du -h -d 2
   ```

5. 显示所有文件和目录的大小并显示总计：
   ```bash
   du -ahc
   ```

## 注意事项
- 默认显示所有子目录的大小
- `-s` 选项只显示总计大小
- `-h` 选项使输出更易读
- 可以结合 `sort` 命令排序，如 `du -h | sort -hr`
- 与 `df` 命令不同，`du` 显示的是文件或目录的实际使用大小，而 `df` 显示的是文件系统的使用情况
