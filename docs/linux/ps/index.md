# ps 命令

## 命令简介
`ps` 命令用于显示当前系统的进程状态，是 "process status" 的缩写。

## 语法
```bash
ps [选项]
```

## 常用选项
- `a`：显示所有终端的进程
- `u`：以用户格式显示进程信息
- `x`：显示没有控制终端的进程
- `e`：显示所有进程
- `f`：以全格式显示进程树
- `l`：以长格式显示进程信息
- `p`：显示指定PID的进程
- `t`：显示指定终端的进程

## 常用组合选项
- `ps aux`：显示所有进程的详细信息
- `ps ef`：显示所有进程的全格式信息
- `ps aux | grep keyword`：搜索指定进程

## 示例
1. 显示当前用户的进程：
   ```bash
   ps
   ```

2. 显示所有进程的详细信息：
   ```bash
   ps aux
   ```

3. 显示进程树：
   ```bash
   ps ef
   ```

4. 搜索指定进程：
   ```bash
   ps aux | grep nginx
   ```

5. 显示指定PID的进程：
   ```bash
   ps p 1234
   ```

## 输出解释
`ps aux` 输出的主要字段：
- `USER`：进程所有者
- `PID`：进程ID
- `%CPU`：CPU使用率
- `%MEM`：内存使用率
- `VSZ`：虚拟内存大小
- `RSS`：物理内存大小
- `TTY`：控制终端
- `STAT`：进程状态
- `START`：启动时间
- `TIME`：CPU时间
- `COMMAND`：命令行

## 注意事项
- `ps` 命令显示的是某个时间点的进程快照
- 要查看动态进程，使用 `top` 命令
- 结合 `grep` 可以搜索特定进程
- 不同系统的 `ps` 命令选项可能略有差异
