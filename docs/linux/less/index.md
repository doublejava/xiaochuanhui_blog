# less 命令

## 命令简介
`less` 命令用于查看文件内容，是一个功能强大的分页查看工具，比 `more` 命令更灵活。

## 语法
```bash
less [选项] 文件
```

## 常用选项
- `-N`：显示行号
- `-i`：忽略大小写搜索
- `-s`：压缩连续的空行
- `-M`：显示更详细的提示信息

## 常用操作
在 `less` 中可以使用以下按键进行操作：
- `空格键`：向下翻一页
- `Enter` 键：向下翻一行
- `b`：向上翻一页
- `k`：向上翻一行
- `h`：显示帮助信息
- `/pattern`：向下搜索 pattern
- `?pattern`：向上搜索 pattern
- `n`：查找下一个匹配项
- `N`：查找上一个匹配项
- `g`：跳转到文件开头
- `G`：跳转到文件末尾
- `q`：退出 `less`

## 示例
1. 查看文件内容：
   ```bash
   less file.txt
   ```

2. 查看文件并显示行号：
   ```bash
   less -N file.txt
   ```

3. 从标准输入读取内容：
   ```bash
   cat file.txt | less
   ```

4. 搜索关键词：
   ```bash
   less file.txt
   /keyword  # 然后按回车搜索
   ```

5. 查看命令输出：
   ```bash
   ps aux | less
   ```

## 注意事项
- `less` 可以向前和向后滚动，而 `more` 只能向前滚动
- `less` 不会一次性加载整个文件，适合查看大文件
- 支持搜索、标记等高级功能
- 可以使用 `less +F` 实时查看日志文件，类似 `tail -f`
