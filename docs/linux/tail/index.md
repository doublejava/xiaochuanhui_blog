# tail 命令

## 命令简介
`tail` 命令用于显示文件的末尾部分内容，默认显示最后10行，常用于查看日志文件。

## 语法
```bash
tail [选项] [文件...]
```

## 常用选项
- `-n`：指定显示的行数，如 `-n 20` 或 `-20`
- `-c`：指定显示的字节数，如 `-c 100`
- `-f`：实时跟踪文件变化，常用于查看日志
- `-F`：与 `-f` 类似，但会跟踪文件的重命名和创建
- `-q`：不显示文件名头
- `-v`：总是显示文件名头

## 示例
1. 显示文件最后10行：
   ```bash
   tail file.txt
   ```

2. 显示文件最后20行：
   ```bash
   tail -n 20 file.txt
   ```
   或
   ```bash
   tail -20 file.txt
   ```

3. 实时跟踪日志文件：
   ```bash
   tail -f /var/log/syslog
   ```

4. 显示文件最后50字节：
   ```bash
   tail -c 50 file.txt
   ```

5. 跟踪文件并显示最后15行：
   ```bash
   tail -n 15 -f /var/log/nginx/access.log
   ```

## 注意事项
- 默认显示最后10行
- `-f` 选项用于实时监控文件变化，按 Ctrl+C 退出
- `-F` 选项比 `-f` 更强大，能处理文件重命名情况
- 可以结合管道使用，如 `ls -l | tail -n 5`
