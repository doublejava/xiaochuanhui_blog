# chmod 命令

## 命令简介
`chmod` 命令用于修改文件或目录的权限，是 "change mode" 的缩写。

## 语法
```bash
chmod [选项] 模式 文件/目录...
```

## 权限表示
- **数字表示法**：
  - r（读）= 4
  - w（写）= 2
  - x（执行）= 1
  - 组合权限：rwx=7, rw-=6, r-x=5, r--=4, -wx=3, -w-=2, --x=1, ---=0
  - 三个数字分别代表：所有者权限、所属组权限、其他用户权限

- **符号表示法**：
  - u：所有者
  - g：所属组
  - o：其他用户
  - a：所有用户
  - +：添加权限
  - -：移除权限
  - =：设置权限

## 常用选项
- `-R`：递归修改目录及其内容的权限
- `-v`：显示修改权限的详细信息
- `-c`：只显示发生变化的文件

## 示例
1. 使用数字表示法设置权限：
   ```bash
   chmod 755 file.txt
   ```

2. 使用符号表示法添加执行权限：
   ```bash
   chmod u+x script.sh
   ```

3. 递归修改目录权限：
   ```bash
   chmod -R 755 directory/
   ```

4. 移除其他用户的写权限：
   ```bash
   chmod o-w file.txt
   ```

5. 设置所有用户的读写权限：
   ```bash
   chmod a=rw file.txt
   ```

## 注意事项
- 只有文件所有者或 root 用户可以修改文件权限
- 目录需要执行权限才能进入
- 可执行文件需要执行权限才能运行
- 数字表示法更简洁，符号表示法更直观
- 使用 `-R` 选项时要小心，避免误改系统文件权限
