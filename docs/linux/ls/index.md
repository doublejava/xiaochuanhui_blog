# ls 命令

## 命令简介
`ls` 命令用于列出目录内容，是 Linux 系统中最常用的命令之一。

## 语法
```bash
ls [选项] [文件/目录]
```

## 常用选项
- `-l`：以长格式显示文件详细信息
- `-a`：显示所有文件，包括隐藏文件（以 . 开头的文件）
- `-h`：以人类可读的格式显示文件大小（如 K、M、G）
- `-t`：按修改时间排序，最新的文件排在前面
- `-r`：反向排序
- `-la`：组合选项，显示所有文件的详细信息

## 示例
1. 列出当前目录内容：
   ```bash
   ls
   ```

2. 列出当前目录的详细信息：
   ```bash
   ls -l
   ```

3. 列出所有文件（包括隐藏文件）：
   ```bash
   ls -a
   ```

4. 列出文件并显示大小：
   ```bash
   ls -lh
   ```

5. 按修改时间排序：
   ```bash
   ls -lt
   ```

## 输出解释
长格式输出包含以下信息：
- 文件类型和权限
- 链接数
- 文件所有者
- 文件所属组
- 文件大小
- 最后修改时间
- 文件名
