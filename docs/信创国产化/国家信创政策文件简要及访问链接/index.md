# 国家信创政策文件简要及访问链接

## 一、政策背景

信创（信息技术应用创新）是国家战略，旨在推动信息技术领域的自主可控，保障国家信息安全。近年来，国家出台了一系列政策文件，指导和推动信创产业发展。

## 二、主要政策文件

### 1. 《国家信息化发展战略纲要》
- **发布时间**：2016年7月
- **发布部门**：中共中央办公厅、国务院办公厅
- **主要内容**：明确了信息化发展的指导思想、基本原则、战略目标和主要任务，提出要加强信息技术自主创新，推动核心技术突破。
- **访问链接**：[国家信息化发展战略纲要](http://www.gov.cn/zhengce/2016-07/27/content_5100093.htm)

### 2. 《“十四五”国家信息化规划》
- **发布时间**：2021年12月
- **发布部门**：中央网络安全和信息化委员会
- **主要内容**：提出了“十四五”时期信息化发展的主要目标和重点任务，强调要加快信创产业发展，提升信息技术自主可控水平。
- **访问链接**：[“十四五”国家信息化规划](http://www.cac.gov.cn/2021-12/27/c_1642619483029388.htm)

### 3. 《关于加快推进国有企业数字化转型工作的通知》
- **发布时间**：2020年9月
- **发布部门**：国务院国资委
- **主要内容**：要求国有企业加快数字化转型，推动信创产品应用，提升核心竞争力。
- **访问链接**：[关于加快推进国有企业数字化转型工作的通知](http://www.sasac.gov.cn/n2588025/n2588124/c20646972/content.html)

### 4. 《网络安全法》
- **实施时间**：2017年6月
- **发布部门**：全国人民代表大会常务委员会
- **主要内容**：明确了网络安全的基本制度和要求，为信创产业发展提供了法律保障。
- **访问链接**：[中华人民共和国网络安全法](http://www.npc.gov.cn/npc/c30834/201611/20161107_200718.html)

### 5. 《数据安全法》
- **实施时间**：2021年9月
- **发布部门**：全国人民代表大会常务委员会
- **主要内容**：规范数据处理活动，保障数据安全，促进数据开发利用，保护个人、组织的合法权益。
- **访问链接**：[中华人民共和国数据安全法](http://www.npc.gov.cn/npc/c30834/202106/20210610_203482.html)

### 6. 《个人信息保护法》
- **实施时间**：2021年11月
- **发布部门**：全国人民代表大会常务委员会
- **主要内容**：保护个人信息权益，规范个人信息处理活动，促进个人信息合理利用。
- **访问链接**：[中华人民共和国个人信息保护法](http://www.npc.gov.cn/npc/c30834/202108/20210820_204798.html)

### 7. 《“十四五”数字经济发展规划》
- **发布时间**：2021年12月
- **发布部门**：国务院
- **主要内容**：提出了“十四五”时期数字经济发展的主要目标和重点任务，强调要推动信创产业发展，提升数字技术创新能力。
- **访问链接**：[“十四五”数字经济发展规划](http://www.gov.cn/zhengce/content/2022-01/12/content_5667482.htm)

### 8. 《关于促进集成电路产业和软件产业高质量发展企业所得税政策的公告》
- **发布时间**：2020年12月
- **发布部门**：财政部、税务总局、发展改革委、工业和信息化部
- **主要内容**：对集成电路产业和软件产业给予企业所得税优惠政策，支持信创产业发展。
- **访问链接**：[关于促进集成电路产业和软件产业高质量发展企业所得税政策的公告](http://www.chinatax.gov.cn/chinatax/n810341/n810755/c5170630/content.html)

## 三、政策影响

这些政策文件为信创产业发展提供了强有力的支持和指导，推动了信创技术的研发和应用，促进了信创生态的建设和完善。对于企业和开发者来说，了解和把握这些政策，有助于更好地参与信创产业发展，抓住市场机遇。

## 四、后续政策趋势

随着信创产业的不断发展，国家将继续出台相关政策，进一步完善信创产业生态，推动信创技术在各个领域的广泛应用。企业和开发者应密切关注政策动态，及时调整发展战略，积极参与信创产业建设。
