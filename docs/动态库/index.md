# 动态库技术专栏

欢迎来到动态库技术专栏！这里记录了Python操作各种动态库的技术文章和实战经验，包括Windows平台的DLL和Linux平台的SO动态库。

## 动态库类型

- [DLL动态库](dll/) - Windows平台的动态链接库
- [SO动态库](so/) - Linux/Unix平台的共享对象库
- [Java NIO操作动态库](java/) - Java使用NIO操作动态库的核心函数

## 核心技术点

- 动态库的加载与卸载
- 函数调用与参数传递
- 数据类型转换
- 错误处理与调试
- 跨平台兼容性
- 性能优化

## 适用场景

- 调用C/C++编写的高性能库
- 访问系统底层API
- 集成第三方SDK
- 保护核心算法
- 实现跨语言调用
